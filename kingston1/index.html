UltraHD 4k Интерфейс UHS-I Speed Class 3 (U3) Video Speed Class V30 обеспечивает пропускную способность карты памяти, достаточную для записи UltraHD 4k видео с повышенной частотой кадров — до 80 МБ/с.

Защита данных Карта памяти Canvas React не боится рентгеновского излучения при досмотре в аэропортах, выдерживает получасовое погружение в воду на глубину до 1 метра, падения, вибрацию, удары и температуру в диапазоне от –25 до +85 °С.

Быстрая работа с отснятым материалом Карта памяти Canvas React способна читать данные со скоростью 100 МБ/с — в 10 раз быстрее, чем стандартные microSD-карты Class

Широкий диапазон емкостей Карты памяти Canvas React представлены в широком диапазоне емкостей — от 32 до 128 ГБ, что позволит подобрать карту, способную вместить воспоминания из любого путешествия.
